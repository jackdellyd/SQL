# SQL

This repository contains SQL scripts, schemas, and migrations.

Initial contents:
- README.md — this file
- LICENSE — MIT license
- .gitignore — common ignores for database files
- schema.sql — a small example schema to get started

How to use

- Add your SQL scripts and organize them into folders (e.g., migrations/, scripts/, examples/).
- Run SQL scripts with your database client, for example:
  - psql -f schema.sql
  - sqlite3 mydb.sqlite < schema.sql
